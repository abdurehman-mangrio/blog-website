# Modern Blog Website

## Description
A modern, responsive blog platform built with React, Vite, Tailwind CSS, React Router, and Lucide icons. Full-featured blog with posts management, categories, search, comments, and user progress tracking.

## Features
- Responsive UI with Tailwind CSS
- Blog posts CRUD (create, read, update, delete)
- Category-based filtering
- Search across posts
- Interactive comment system
- Recent posts sidebar
- Progress bar for reading
- React Context for state management
- Routing with React Router DOM
- Professional design with smooth animations

## Tech Stack
**Frontend:**
- React 18+
- Vite (build tool)
- Tailwind CSS (styling)
- React Router DOM (routing)
- Lucide React (icons)
- date-fns (date formatting)
- PostCSS & Autoprefixer

**Development Tools:**
- ESLint
- TypeScript support
- Vite HMR

## Prerequisites
- Node.js 18+
- npm or yarn

## Installation
```bash
git clone https://github.com/abdurehman-mangrio/blog-website.git
cd blog-website
npm install
npm run dev
```

Open http://localhost:5173

## Usage
- Browse home page with recent posts
- View individual blog posts with comments
- Filter by category
- Search posts
- Create/edit posts (admin-like)
- Responsive on all devices
